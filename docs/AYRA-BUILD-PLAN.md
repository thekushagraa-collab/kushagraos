## PHASE B — Agent Engine + Style Lock (foundation for all agents)
**Goal:** one engine + one branded artifact system that every agent reuses.
- Extend the runner (`api/_lib/runCore.ts` + `src/apps/runApp.ts`) into an **agent framework**: add agent
  ids `scout · closer · muse · strategist · concierge · analyst`, each with a strict JSON schema, a tool
  system-prompt, a `coerce()`, and a realistic fallback sample (same pattern as flow/atlas/forge).
- Build the **ArtifactFrame** (Style Lock §11): letterhead (AYRA mark · "Generated by KushagraOS · AYRA" ·
  timestamp) + footer CTA ("Built by Kushagra with AYRA — want this live for your business?"), locked
  fonts/tokens, per-agent signature accent. **Model returns JSON only; React owns all styling.**
- Branded renderers: `LeadSheet` (Scout), `EmailSequence` (Closer), `ContentPack` (Muse), `CampaignBrief`
  (Strategist), `ReplyTriage` (Concierge), `DealBrief` (Analyst).
- **DoD:** each agent endpoint returns valid structured JSON → renders in ArtifactFrame; tests assert locked
  fonts (Fraunces/General Sans/Geist Mono) + token colors + `fallback:false` with key, graceful sample without.
